Trip Planner Público

Roteiros com busca online, sem login Google.

Busca destinos reais no OpenStreetMap/Nominatim e monta o roteiro consultando atrações, restaurantes e cafés via Overpass API. Salva localmente no navegador.

Criar viagem

Adicione países, estados ou cidades. Cada item precisa ser selecionado na lista de resultados para evitar erro de acento ou digitação.

Mapa & salvos

Os destinos selecionados aparecem no mapa. Os roteiros ficam no localStorage do navegador.

Sem chave e sem login: usa serviços públicos do ecossistema OSM. Para muitos usuários simultâneos, o ideal é migrar para servidor próprio ou API paga.
